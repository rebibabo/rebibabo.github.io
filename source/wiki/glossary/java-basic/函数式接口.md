# 函数式接口

[[wiki/glossary/java-basic/index|返回词汇表]]

## 定义

函数式接口（Functional Interface）是只包含一个抽象方法的接口，Lambda 表达式只能赋值给函数式接口类型的变量。可以用 `@FunctionalInterface` 注解让编译器校验接口是否符合规范。

## 上下文

Java 在 `java.util.function` 包中预置了四大核心函数式接口：`Function<T,R>`（转换，T 入 R 出）、`Predicate<T>`（判断，返回 boolean）、`Consumer<T>`（消费，无返回值）、`Supplier<T>`（生产，无入参有返回值）。此外还有 `BiFunction`、`BiPredicate`、`BiConsumer` 等双参数变体，以及 `UnaryOperator`、`BinaryOperator` 等同类型输入输出的简化版本。函数式接口还支持通过 `andThen`、`compose`、`and`、`or`、`negate` 等默认方法组合成更复杂的逻辑，避免嵌套 Lambda。
