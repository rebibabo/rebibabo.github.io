# 方法引用

[[wiki/glossary/java-basic/index|返回词汇表]]

## 定义

方法引用（Method Reference）是 Lambda 表达式的简写形式，当 Lambda 体仅仅是调用一个已有的方法时，可以用 `类名/对象::方法名` 的语法替代，代码更简洁可读。

## 上下文

方法引用有四种形式：静态方法引用 `ClassName::staticMethod`（如 `Integer::parseInt`）；特定对象的实例方法引用 `instance::method`（如 `System.out::println`）；类的实例方法引用 `ClassName::instanceMethod`（如 `String::compareToIgnoreCase`，第一个参数成为方法调用者）；构造方法引用 `ClassName::new`（如 `ArrayList::new`）。方法引用不仅仅是为了少写几个字符，更重要的是它直接传达了"这里就是调那个方法"的语义，比 Lambda 表达式的意图更明确。与 Lambda 一样，方法引用也只能赋值给函数式接口。
