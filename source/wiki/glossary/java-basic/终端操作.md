# 终端操作

[[wiki/glossary/java-basic/index|返回词汇表]]

## 定义

终端操作（Terminal Operation）是 Stream 管道中触发整个流水线执行并产生最终结果的操作。流在终端操作执行后即被消费，不可再次使用。

## 上下文

常用终端操作分几类：收集类——`collect(Collectors.toList())` 将结果收集为集合，是最常用的终端操作；遍历类——`forEach` 对每个元素执行操作；聚合类——`count`（计数）、`reduce`（归约，将流中所有元素合并为一个值，如求和求积）、`min`/`max`（最值）；查找类——`findFirst`（第一个元素）、`findAny`（任意一个，并行流下更快）；匹配类——`anyMatch`（任一满足）、`allMatch`（全部满足）、`noneMatch`（全不满足）；转换类——`toArray` 转为数组。`collect` 配合 `Collectors` 工具类可以实现分组、分区、拼接、统计等丰富的数据收集需求。
