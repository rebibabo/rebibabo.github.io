# 注解

[[wiki/glossary/java-basic/index|返回词汇表]]

## 定义

注解（Annotation）是一种特殊的接口（`@interface`），用于给代码附加元数据标记，本身不包含业务逻辑，真正的逻辑由读取注解的代码（通常通过反射）来执行。

## 上下文

注解编译后变为继承 `java.lang.annotation.Annotation` 的接口，注解属性用方法形式定义（如 `String value() default ""`），支持的类型包括基本类型、String、Class、枚举、其他注解及其数组。Java 内置注解包括 `@Override`（检查方法重写）、`@Deprecated`（标记过时）、`@SuppressWarnings`（抑制编译器警告）、`@FunctionalInterface`（检查函数式接口）。注解配合反射是 Spring 等框架的基础：`@Autowired` 通过反射扫描字段并注入依赖，`@RequestMapping` 通过反射建立 URL 映射，`@Transactional` 通过动态代理添加事务逻辑。Lombok 的 `@Data` 是例外，不走反射而是编译期通过注解处理器（APT）直接生成字节码。
