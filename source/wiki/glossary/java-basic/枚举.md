# 枚举

[[wiki/glossary/java-basic/index|返回词汇表]]

## 定义

枚举（enum）是一种特殊的类，编译后成为继承 `java.lang.Enum` 的 final 类，每个枚举值是一个 static final 的自身类型实例，构造方法为 private，实例数量在编译期固定。

## 上下文

枚举解决了 `static final int` 常量的三个痛点：类型不安全（可传入任意 int）、可读性差（日志中只显示数字）、没有命名空间（常量名可能冲突）。枚举常用方法：`name()` 返回常量名、`ordinal()` 返回声明顺序（从 0）、`values()` 返回所有实例数组、`valueOf()` 字符串转枚举。枚举比较直接用 `==` 而非 `equals`，因为每个枚举值是全局唯一的单例。枚举可以有字段、构造方法和自定义方法，支持实现接口实现策略模式（每个枚举值给出不同的方法实现）、实现线程安全的单例（Effective Java 推荐，天然防反射攻击和反序列化攻击）。枚举专用集合 EnumSet（基于位向量）和 EnumMap（基于数组）比 HashSet/HashMap 性能更好。
