# 栈

[[wiki/glossary/java-basic/index|返回词汇表]]

## 定义

虚拟机栈（VM Stack）是 JVM 中线程私有的内存区域，每个方法调用会压入一个栈帧（Stack Frame），方法执行完毕弹出，用于存储局部变量、操作数栈和返回地址。

## 上下文

每个栈帧包含四个部分：**局部变量表**（存放基本类型的值和对象引用）、**操作数栈**（方法执行时的中间计算结果）、**动态链接**（指向方法区中该方法的引用）、**返回地址**（方法执行完后回到哪里）。局部变量的基本类型值直接存储在栈上，对象引用存在栈上但对象实例在堆上。栈可能抛出两种异常：`StackOverflowError`（栈深度超限，通常是无限递归）和 `OutOfMemoryError`（创建线程过多，每个线程都要分配栈空间）。可通过 `-Xss` 参数设置每个线程的栈大小，但治标不治本，递归过深应检查终止条件或改为迭代。
