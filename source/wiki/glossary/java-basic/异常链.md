# 异常链

[[wiki/glossary/java-basic/index|返回词汇表]]

## 定义

异常链（Chained Exceptions）是 Java 异常机制中通过 `Throwable` 构造函数的 `cause` 参数，将底层异常作为上层异常的原因串联起来的能力，使得排查问题时可以追溯完整错误链路。

## 上下文

在分层架构中，底层异常（如 `IOException`）对上层调用者通常没有意义，需要用业务异常包装。关键做法是**保留原始异常链**：`throw new RuntimeException("处理失败", e)` —— 将原始异常作为 cause 传入。如果丢失 cause（`throw new RuntimeException("处理失败")`），排查问题时只能看到"处理失败"而完全不知道根因。捕获端可以通过 `e.getCause()` 获取原始异常，`printStackTrace()` 会以 "Caused by:" 的层级形式递归打印完整链路。这是异常处理中最基本也最容易忽视的最佳实践。
