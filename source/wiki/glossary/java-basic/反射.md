# 反射

[[wiki/glossary/java-basic/index|返回词汇表]]

## 定义

反射（Reflection）是 Java 在运行时动态获取类信息、创建对象、调用方法和访问字段的能力，核心 API 包括 Class、Constructor、Field、Method。

## 上下文

获取 Class 对象的三种方式：`Class.forName("全限定名")`（运行时动态加载，框架常用）、`类名.class`（编译期已知类型）、`对象.getClass()`（从已有对象获取）。反射通过 Constructor 创建对象，Field 读写字段，Method 调用方法，配合 `setAccessible(true)` 可以突破 private 封装限制。`getXxx` 系列只获取 public 成员（包括继承的），`getDeclaredXxx` 获取所有成员（含 private，但不含继承的）。反射比直接调用慢 10-50 倍，因为需要绕过编译期优化、进行安全检查，优化方式包括缓存 Method/Field 对象和 `setAccessible(true)`。日常业务代码极少需要直接写反射，但它是读懂 Spring、Jackson、MyBatis 等框架源码的前提——Spring 通过反射实现依赖注入，Jackson 通过反射实现序列化。
