# 集合框架

[[wiki/glossary/java-basic/index|返回词汇表]]

## 定义

Java Collections Framework 是 Java 中用于存储和操作一组对象的标准 API 体系，顶层分两条独立线：`Collection`（存储单个元素，下分 List、Set、Queue）和 `Map`（存储键值对）。

## 上下文

集合框架是日常开发中使用频率最高的 API，没有之一。核心接口包含 List（有序可重复）、Set（无序不可重复）、Queue（FIFO 队列）、Map（键值映射），各接口有多个实现类适应不同场景。选型的关键在于理解底层数据结构和时间复杂度差异：需要有序列表首选 ArrayList，去重用 HashSet，键值映射用 HashMap。注意多线程场景下需要 ConcurrentHashMap 而非 HashMap。
