# 字符流

[[wiki/glossary/java-basic/index|返回词汇表]]

## 定义

字符流（Reader/Writer）是 Java I/O 体系中处理文本数据的抽象基类，以字符（char）为最小读写单位，内置字符编码转换，适用于 .txt、.csv、.json、.xml 等文本文件的读写。

## 上下文

`FileReader` / `FileWriter` 使用系统默认编码，跨平台容易产生乱码，不推荐直接使用。正确的做法是通过桥接流 `InputStreamReader` / `OutputStreamWriter` 将字节流转换为字符流并显式指定编码（如 `StandardCharsets.UTF_8`）。实际开发中常配合 `BufferedReader`（提供 `readLine()` 方法逐行读取）或 `BufferedWriter`（提供 `newLine()` 写入换行符）装饰器提升性能。`PrintWriter` 进一步提供 `println`、`printf` 等便捷写入方法。选择规则：文本文件用字符流，二进制文件（图片、音频、视频等）用字节流；不确定时用字节流总不会错。
