# Bid Request（竞价请求）

[[wiki/glossary/programmatic-ads/index|返回词汇表]]

## 定义

Bid Request 是 SSP/ADX 发给 DSP 的竞价请求消息，描述"这里有一个广告展示机会，你要不要出价、出多少"。它是programmatic-ads竞价链路的第一个消息环节。

## 上下文

一个 Bid Request 的典型 JSON 结构包含：请求 ID、展示机会（Impression）数组（含广告位信息、底价、支持的素材格式）、设备信息、用户信息（或其去标识化标识符）、以及可选的 Deal ID（如果是私有交易）。

DSP 收到 Bid Request 后需要在 100ms 级别内完成用户判断、出价计算并返回 Bid Response，否则会被 ADX 超时丢弃。
