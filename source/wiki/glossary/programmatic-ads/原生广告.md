# Native Ads（原生广告）

[[wiki/glossary/programmatic-ads/index|返回词汇表]]

## 定义

Native Ads 是一种广告形式：广告的样式、风格、交互与页面内容自然融合，而不是以横幅、弹窗等明显外来的形式出现。信息流广告（Feeds Ads）是最常见的原生广告形态。

## 上下文

在程序化广告中，Native Ads 通过 OpenRTB Native Ads Spec 标准化：Bid Request 中描述广告位需要的素材组件（标题、图片、描述文字等），Bid Response 中各组件分别填充内容，最后由 SSP 按媒体方的 UI 风格渲染。

原生广告的关键优势是用户体验更好——广告不打断用户的浏览流程，点击率和转化率通常高于传统 Banner。
