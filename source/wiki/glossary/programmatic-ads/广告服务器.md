# Ad Server（广告服务器）

[[wiki/glossary/programmatic-ads/index|返回词汇表]]

## 定义

Ad Server 是存储广告素材、决定展示哪条广告、记录投放数据的服务系统。它是广告投放链路的"最后一公里"——前面的竞价决定了"谁赢了"，Ad Server 负责"把广告真正展示出去"。

## 上下文

Ad Server 分为两类：

- **Third-party Ad Server**（广告主侧）：广告主用来统一管理跨平台投放、收集效果数据。典型代表：Google Campaign Manager。
- **Publisher Ad Server**（媒体侧）：媒体方用来管理广告位排期、决定每次展示投哪条广告。

Ad Server 的核心价值是**独立计数**——它不参与竞价，只负责记录真实曝光和点击，是结算对账的"裁判"。
