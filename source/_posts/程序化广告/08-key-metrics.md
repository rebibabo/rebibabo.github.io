前几篇讲完了所有角色、所有交易模式，程序化广告的"骨架"和"规则"都已经完整。

但还有一个绕不开的核心问题：**怎么衡量"投得好不好"？**

广告主每天面对一堆问题：这次活动效果如何？预算花对地方了吗？哪个渠道 ROI 最高？一个用户看了 5 个广告才下单，功劳算谁的？

这一篇讲考核指标，程序化广告的"成绩单"。

<!-- more -->

## 一、先看完整的指标体系

理解考核指标最好的方式，是把它放进**投放漏斗**里看。

广告投放本质上是一个漏斗：从最广的"曝光"开始，逐层向下收窄，直到最终的"转化"和"收益"。每一层都有自己的核心指标。

每往下一层，**人数大幅减少**：

可能 100 万次曝光中有 1 万次点击（CTR 1%），1 万次点击里有 100 次转化（CVR 1%），最终 100 次转化带来 5 万元收入。

理解这个漏斗，再看具体指标就清晰得多。

下面按四层逐一展开。

## 二、曝光层指标

曝光层关心的核心问题是：**广告被多少人看到了？看到的质量如何？**

### 2.1 Impression（曝光）

**Impression（曝光）** 是最基础的指标，指广告被加载并展示的次数。

注意：**只要广告被加载就算 1 次 Impression**，不管用户有没有真正看到（比如广告在屏幕外、被遮挡、加载后用户立刻关页面），都计为曝光。

这就是为什么后面会出现"Viewability（可视率）"这个修正指标。

### 2.2 CPM（千次曝光成本）

**CPM（Cost Per Mille，千次曝光成本）** 是程序化广告里最基础的计费模式之一。

```
CPM = (总花费 / 总曝光数) × 1000
```

举例：广告主花了 5000 元，获得了 100 万次曝光。
CPM = (5000 / 1,000,000) × 1000 = **5 元**

也就是说，每 1000 次曝光花了 5 元。

**适用场景**：品牌曝光、新品发布——不需要用户立刻行动，只需要让大家知道。

### 2.3 eCPM（有效千次曝光成本）

**eCPM（Effective CPM，有效千次曝光成本）** 是 CPM 的"通用化"版本，把所有计费模式（CPM、CPC、CPA）都换算成"等效的千次曝光收益"。

> 💡 这里先简单了解两个缩写：**CPC（Cost Per Click，单次点击成本）** 和 **CPA（Cost Per Action，单次转化成本）**。它们的详细含义会在第三节和第四节展开讲。

```
eCPM = (总收益 / 总曝光数) × 1000
```

为什么需要 eCPM？因为媒体方接入多个广告源，有的按 CPM 计费，有的按 CPC 计费，有的按 CPA 计费。**为了对比哪个广告源更赚钱，需要换算到统一的量纲上**。

举例：

| 广告源 | 计费方式 | 总收益 | 总曝光数 | eCPM |
|--------|---------|--------|---------|------|
| 源 A | CPM | 5000 元 | 100 万次 | 5 元 |
| 源 B | CPC | 8000 元 | 100 万次 | 8 元 |
| 源 C | CPA | 6000 元 | 100 万次 | 6 元 |

**虽然计费方式不同，但 eCPM 直接可比**——源 B 最赚钱。

这就是为什么 eCPM 是**媒体方最重要的指标**：它决定了"哪一千次曝光最值钱"。

### 2.4 vCPM（可视千次曝光成本）

**vCPM（Viewable CPM，可视千次曝光成本）** 解决了 CPM 的一个核心痛点：**很多"曝光"根本没被用户看到**。

```
vCPM = (总花费 / 可视曝光数) × 1000
```

广告主越来越倾向用 vCPM 而不是 CPM 付费——因为只为"真正被看到的曝光"付费才合理。

### 2.5 Viewability（可视率）

**Viewability（可视率）** 是衡量"曝光质量"的核心指标。

IAB（Interactive Advertising Bureau，互动广告局）定义的标准：

- **展示广告**：广告至少 **50% 的像素**在用户屏幕内，**持续 1 秒以上**
- **视频广告**：广告至少 **50% 的像素**在用户屏幕内，**持续 2 秒以上**

满足条件的曝光才算"可视曝光"。

```
Viewability = (可视曝光数 / 总曝光数) × 100%
```

行业基准：

| 媒体类型 | Viewability 基准 |
|---------|-----------------|
| 优质媒体 | > 70% |
| 一般媒体 | 50% - 70% |
| 低质媒体 | < 50% |

如果一个广告源 Viewability 不到 50%，广告主会直接拉黑。

## 三、点击层指标

点击层关心：**多少人对广告感兴趣到愿意点击？**

### 3.1 Click（点击）

**Click（点击）** 指用户点击了广告。看似简单，但实际有不少坑：

- **误点击**：用户手指划过屏幕时不小心点到，这种点击对广告主没有价值
- **机器点击**：自动化脚本生成的点击（典型的 IVT）
- **重复点击**：同一用户短时间内多次点击，通常只算 1 次

成熟的 DSP 会自动过滤这些"无效点击"。

### 3.2 CTR（点击率）

**CTR（Click-Through Rate，点击率）** 是衡量广告"吸引力"的核心指标。

```
CTR = (点击数 / 曝光数) × 100%
```

行业基准（很粗略的参考）：

| 广告类型 | CTR 基准 |
|---------|---------|
| 展示广告（Banner） | 0.1% - 0.5% |
| 信息流广告（Native） | 0.5% - 2% |
| 视频广告（贴片） | 1% - 3% |
| 搜索广告 | 3% - 5% |

CTR 太低说明创意不吸引人或定向不准；CTR 太高（比如展示广告 > 3%）反而可能有问题（误点击或假流量）。

### 3.3 CPC（单次点击成本）

**CPC（Cost Per Click，单次点击成本）** 是按点击计费的核心指标。

```
CPC = 总花费 / 点击数
```

举例：广告主花了 5000 元，获得 1000 次点击。
CPC = 5000 / 1000 = **5 元**

也就是说，每带来一个用户点击花了 5 元。

**适用场景**：拉新引流、电商导流——关注的是"把人带过来"。

## 四、转化层指标

转化层关心：**点击后，多少人真的完成了广告主期望的行动？**

这是广告主**最在意**的一层。

### 4.1 Conversion（转化）

**Conversion（转化）** 指用户完成了广告主预设的目标行动。

"转化"的定义因广告主而异：

| 广告主类型 | 典型转化定义 |
|----------|-------------|
| 电商 | 下单、付款 |
| 游戏 | 下载、注册、首次付费 |
| 教育 | 留资、试听课报名 |
| SaaS | 注册账号、开通试用 |
| 金融 | 申请信用卡、开通账户 |
| 内容平台 | 注册、关注、订阅 |

**注意"转化"是个层级概念**——不同阶段的转化价值不同：

```
点击落地页 → 注册 → 试用 → 付费 → 复购
（浅层转化）  ←—————————————————→  （深层转化）
```

广告主通常会设置多个转化目标，分别追踪。

### 4.2 CVR（转化率）

**CVR（Conversion Rate，转化率）** 衡量"点击到转化"的效率。

```
CVR = (转化数 / 点击数) × 100%
```

CVR 直接反映**落地页质量**和**用户匹配度**。

CVR 低通常有几个原因：

- 落地页加载慢
- 落地页内容和广告承诺不符
- 注册流程太复杂
- 定向不准（吸引到错误的人）

### 4.3 CPA（单次转化成本）

**CPA（Cost Per Action，单次转化成本）** 是广告主**最常用的考核指标**。

```
CPA = 总花费 / 转化数
```

举例：广告主花了 5000 元，获得 100 次转化。
CPA = 5000 / 100 = **50 元**

也就是说，每带来一个有效转化花了 50 元。

广告主通常会设置 **目标 CPA**：

> "我希望每个新用户的获取成本不超过 50 元，超过就调整策略。"

DSP 会根据目标 CPA 自动优化出价——这就是 **oCPC（Optimized CPC）** 和 **oCPM（Optimized CPM）** 的核心机制。

## 五、效果层指标

效果层是**最终极的考核**：**广告主投入这些钱，最后赚了多少？**

### 5.1 ROI（投资回报率）

**ROI（Return on Investment，投资回报率）** 是最经典的财务指标。

```
ROI = (收益 - 成本) / 成本 × 100%
```

举例：广告主投入 10 万元广告费，带来了 30 万元收入。
ROI = (30 - 10) / 10 × 100% = **200%**

ROI > 0 说明赚钱，ROI > 100% 说明翻倍。

但 ROI 在广告行业用得不算多，因为"收益"很难精确算（要扣除各种成本）。更常用的是 ROAS。

### 5.2 ROAS（广告支出回报率）

**ROAS（Return on Ad Spend，广告支出回报率）** 是 ROI 的"广告版"，只看"广告带来的收入"。

```
ROAS = 广告带来的收入 / 广告花费
```

举例：广告主花了 1 万元广告费，带来了 5 万元订单收入。
ROAS = 50000 / 10000 = **5**

也就是说，**每投 1 元广告，带回 5 元收入**。

行业经验：

| ROAS 区间 | 评价 |
|----------|------|
| < 1 | 亏损 |
| 1 - 2 | 勉强保本 |
| 2 - 4 | 健康 |
| > 4 | 优秀 |

不同行业的 ROAS 标准不同——奢侈品行业 ROAS 5+ 才算及格，快消品行业 ROAS 2 就很不错。

### 5.3 LTV（用户生命周期价值）

**LTV（Lifetime Value，用户生命周期价值）** 是更深入的指标，考虑**用户长期能带来多少价值**。

```
LTV = 平均订单价值 × 复购频次 × 用户留存时长
```

举例：

- 一个电商用户平均每单 200 元
- 平均每年复购 4 次
- 平均留存 3 年
- LTV = 200 × 4 × 3 = **2400 元**

**为什么 LTV 比 CPA 更重要？**

假设广告主用 CPA 100 元获取了一个用户：

- 只看 CPA：花了 100 元，似乎"成本不低"
- 看 LTV：这个用户未来能带 2400 元收入，**ROI 24 倍**

所以**成熟的广告主用 LTV 来评估"获客成本是否值得"**：

```
只要 LTV > CPA × 3，就值得继续投放
```

这也是为什么很多公司愿意为新用户"亏钱获客"——只要 LTV 够高，长期能赚回来。

## 六、品牌层指标

前面五层都是"效果广告"的指标——重点是"转化和收益"。

但还有一类广告**主要看品牌效果**，不追求即时转化。比如可口可乐的电视广告、奢侈品的开屏广告，这些广告的目标不是"立刻让你下单"，而是**让你记住这个品牌**。

这类广告需要另一套指标。

### 6.1 Reach（触达人数）

**Reach（触达人数）** 指广告触达了多少**不重复**的用户。

注意和 Impression 的区别：

| 指标 | 含义 |
|------|------|
| Impression（曝光数） | 广告展示了多少次（可能重复） |
| Reach（触达人数） | 多少不重复的人看到了广告 |

举例：100 万次曝光，可能只触达了 30 万个不同的人（平均每人看了 3.3 次）。

品牌广告主更关心 Reach——**让更多人知道**比"让一部分人反复看"重要。

### 6.2 Frequency（触达频次）

**Frequency（触达频次）** 是 Reach 的配套指标。

```
Frequency = Impression / Reach
```

举例：100 万次曝光 / 30 万人触达 = **3.3 次**

也就是说，平均每个用户看了 3.3 次广告。

业界经验：

- 频次太低（< 2 次）：用户记不住
- 频次合适（3-7 次）：记忆深刻又不烦人
- 频次太高（> 10 次）：用户开始反感，伤害品牌

成熟的品牌广告主会做"**频次控制**"——限制同一用户在一段时间内的最大曝光次数。

### 6.3 Brand Lift（品牌提升度）

**Brand Lift（品牌提升度）** 是衡量品牌广告效果的"金标准"。

做法：通过问卷调查对比"看过广告的人"和"没看过广告的人"在几个维度的差异。

| 维度 | 衡量内容 |
|------|---------|
| 品牌认知（Awareness） | 知道这个品牌的比例 |
| 品牌好感（Favorability） | 对品牌的好感度 |
| 购买意向（Purchase Intent） | 未来购买的可能性 |
| 广告记忆（Ad Recall） | 还能回忆起广告的比例 |

```
Brand Lift = (看过广告组指标 - 未看过广告组指标) / 未看过广告组指标
```

举例：看过广告的人中 60% 知道这个品牌，没看过的人中 40% 知道。
Brand Lift = (60% - 40%) / 40% = **50%**

也就是说，广告让品牌认知度**提升了 50%**。

Brand Lift 数据通常由第三方调研公司（如 Nielsen、Kantar）或大型媒体平台（Google、Meta）提供。

## 七、归因（Attribution）：最容易引起争议的话题

讲完了所有"直接指标"，我们必须单独谈一个话题——**归因**。

归因不是单一指标，而是一个"**功劳分配机制**"。它决定了所有效果指标（CPA、ROAS、CVR）的统计口径——**没有它，所有数字都没法解读**。

### 7.1 归因是什么

**Attribution（归因）** 要回答的问题是：

> 一个用户在最终转化前，可能接触过多个广告。这次转化的"功劳"，应该算给哪个广告？

举例：

```
用户行为路径：
Day 1：在抖音看到信息流广告  →  没行动
Day 3：在朋友圈看到广告      →  点击了，但没下单
Day 5：在百度搜索品牌词       →  点击了搜索广告，但没下单
Day 7：直接打开 APP          →  下单了
```

这次下单的"功劳"算给谁？

- 算给最后一次接触（搜索广告）？
- 算给第一次接触（抖音曝光）？
- 平均分给每一次接触？
- 按某种权重算？

**不同的归因模型，结果完全不同**——可能差异 10 倍以上。

### 7.2 6 种主流归因模型

#### 模型 1：Last Click（最后点击归因）

**规则**：把功劳 100% 归给"用户转化前最后点击的那个广告"。

| 优点 | 缺点 |
|------|------|
| 简单直接 | 忽视前期"种草"贡献 |
| 容易实现 | 偏向漏斗底部渠道（搜索、直访） |

行业现状：Google Analytics 默认就是 Last Click。这是最常见但也最有偏见的模型。

#### 模型 2：First Click（首次点击归因）

**规则**：把功劳 100% 归给"用户第一次接触的那个广告"。

| 优点 | 缺点 |
|------|------|
| 重视品牌曝光 | 忽视后期临门一脚 |
| 适合品牌广告评估 | 偏向漏斗顶部渠道（信息流、视频） |

#### 模型 3：Linear（线性归因）

**规则**：所有接触过的广告**平均分**功劳。

举例：用户在转化前接触了 4 个广告，每个广告分 25% 功劳。

| 优点 | 缺点 |
|------|------|
| 公平、简单 | 没考虑"哪个广告作用更大" |

#### 模型 4：Time Decay（时间衰减归因）

**规则**：**越接近转化时间的广告，分到的功劳越多**。

举例：用户在转化前 7 天、5 天、3 天、1 天分别接触 4 个广告，最近的那个分最多。

| 优点 | 缺点 |
|------|------|
| 符合"临门一脚更重要"的直觉 | 仍然偏向漏斗底部 |
| 平衡了时间因素 | 忽视品牌长期影响 |

#### 模型 5：Position-Based（位置归因，又叫 U 型归因）

**规则**：第一次和最后一次接触各分 **40%**，中间所有接触平均分 **20%**。

| 优点 | 缺点 |
|------|------|
| 兼顾"开头种草"和"结尾促单" | 中间渠道被压得太低 |
| 适合品牌+效果混合投放 | 模型有点武断 |

#### 模型 6：Data-Driven Attribution（数据驱动归因）

**规则**：用**机器学习模型**动态分配功劳，基于大量历史数据学习"每个接触点的真实贡献"。

| 优点 | 缺点 |
|------|------|
| **最精准** | 需要大量数据 |
| 自适应不同行业 | 黑盒，难解释 |
| Google、Meta 等大平台主推 | 中小广告主难以实施 |

### 7.3 不同模型对预算分配的影响

归因模型不是"学术问题"，而是**直接影响真金白银的预算分配**。

举一个具体例子。

某电商在 4 个渠道投放，本月共 100 次转化。不同归因模型下，4 个渠道分到的"功劳"完全不同：

| 渠道 | Last Click | First Click | Linear | Time Decay |
|------|-----------|-------------|--------|-----------|
| 信息流（种草） | 10 次 | **50 次** | 25 次 | 15 次 |
| 朋友圈（深化） | 20 次 | 25 次 | 25 次 | 25 次 |
| 搜索广告（决策） | 30 次 | 15 次 | 25 次 | 30 次 |
| 直访（成交） | **40 次** | 10 次 | 25 次 | **30 次** |

按 Last Click：广告主会**减少信息流投放，加大直访**——但这是错的，因为信息流是"种草"的源头。

按 First Click：广告主会**减少搜索广告**——但这也是错的，因为搜索广告是"临门一脚"。

**选错归因模型，预算就会分错地方**。

### 7.4 业界最佳实践

成熟的广告主**不依赖单一归因模型**，而是用以下组合：

1. **主用 Data-Driven Attribution**（如果数据量足够）
2. **辅以 Last Click 和 First Click 对比**（看两端差异）
3. **对品牌广告用 Brand Lift**（不用转化归因）
4. **对效果广告用 ROAS**（结合 Data-Driven）

**归因不是寻找"唯一真相"，而是寻找"对决策最有用的视角"**。

## 八、媒体方视角：另一套指标

前面讲的都是**广告主关心的指标**。媒体方关心的是另一套——它们更关心"**怎么把流量卖好**"。

### 8.1 Fill Rate（填充率）

**Fill Rate（填充率）** 是衡量"广告位是否被填满"的指标。

```
Fill Rate = (实际展示的广告数 / 总广告请求数) × 100%
```

举例：媒体有 100 万次广告请求，实际展示了 80 万次（剩下 20 万次因为"没人买"或"出价低于底价"而空着）。
Fill Rate = **80%**

**填充率每提升 5%，媒体收入直接提升 5%**——空着的广告位是纯粹的浪费。

### 8.2 eCPM（再次出现）

eCPM 我们在曝光层讲过，但它**对媒体方更重要**——
它直接决定"每千次曝光能赚多少钱"。

媒体方的优化目标，本质上就是**最大化 eCPM**。

### 8.3 IVT Rate（无效流量率）

**IVT Rate（Invalid Traffic Rate，无效流量率）** 衡量"流量中有多少是假的"。

```
IVT Rate = (无效流量数 / 总流量数) × 100%
```

IVT 太高，广告主就不会再买这家媒体的流量。**媒体方必须控制 IVT Rate 在合理范围内**（通常 < 5%）。

### 8.4 Brand Safety Rate（品牌安全率）

**Brand Safety Rate（品牌安全率）** 衡量"广告环境是否安全"。

不合适的内容环境（暴力、色情、争议政治）会让广告主拉黑这家媒体。媒体方必须主动屏蔽这些内容。

## 九、广告主怎么选指标

最后回答一个实战问题：**广告主在不同投放目标下，应该看哪些指标？**

| 投放目标 | 核心指标 | 辅助指标 | 不重要 |
|---------|---------|---------|--------|
| **品牌曝光** | Reach, Viewability, Brand Lift | CPM, Frequency | CPA, ROAS |
| **拉新引流** | CTR, CPC | CPM, Viewability | ROAS（短期） |
| **效果转化** | CPA, ROAS | CVR, CTR | Reach |
| **长期获客** | LTV, ROAS | CPA | CPM |
| **促活复购** | ROAS, 复购率 | CTR | Reach |

**一个常见的错误**：用同一套指标考核所有投放。

比如品牌广告也用 ROAS 考核——结果发现"ROAS 很低，预算被砍掉"，但其实品牌广告本来就不该追求短期 ROAS，应该看 Brand Lift。

**指标选错，决策就错**。

## 十、整体总结

回到一开始的漏斗图：

```
曝光层 → 点击层 → 转化层 → 效果层
CPM/Viewability → CTR/CPC → CVR/CPA → ROAS/LTV
```

加上**品牌层**（Reach/Frequency/Brand Lift）和**归因**（决定如何统计），就是程序化广告完整的指标体系。

几个核心观点收尾：

**1. 没有"万能指标"**

不同投放目标看不同指标。品牌投放看 Reach/Brand Lift，效果投放看 ROAS/LTV。

**2. 归因决定一切**

所有效果指标（CPA、ROAS、CVR）都依赖归因模型。**归因变了，数字就变了**。

**3. LTV 比 CPA 更重要**

成熟的广告主不只看"获客成本"，更看"用户长期价值"。

**4. 媒体方和广告主用不同的指标**

广告主关心 CPA/ROAS，媒体方关心 eCPM/Fill Rate。理解这种差异，才能理解他们的博弈关系。
