# 原始类型（Primitive Types）

[[wiki/glossary/java-basic/index|返回词汇表]]

## 定义

Java 中有且只有 8 种原始类型（基本数据类型），分为四大类：整型（byte/1字节、short/2字节、int/4字节、long/8字节）、浮点（float/4字节、double/8字节）、字符（char/2字节，Unicode 无符号）和布尔（boolean，true/false，长度未规定）。

## 上下文

原始类型存储在栈上（局部变量）或直接嵌在对象内存布局中，不经过堆分配，访问速度优于包装类型。整型字面量默认是 `int`，浮点字面量默认是 `double`，这是常见的编译错误来源。

`byte`、`short`、`char` 之间参与算术运算时，结果一律先自动提升为 `int`，这是 Java 的类型提升规则。原始类型不能用于泛型参数，必须用对应的包装类型。boolean 不能和任何其他类型互转（不同于 C/C++）。
