# 方法区

[[wiki/glossary/java-basic/index|返回词汇表]]

## 定义

方法区是 JVM 中线程共享的内存区域，存储已加载的类信息、常量池、静态变量和 JIT 编译后的代码缓存。

## 上下文

Java 7 及之前方法区以**永久代（PermGen）**实现，位于 JVM 内存中，大小有限容易 OOM。Java 8 起改为**元空间（Metaspace）**，使用本地内存（Native Memory），默认不限大小，从根本上解决了 PermGen OOM 问题。元空间溢出场景主要是动态生成大量类（如 CGLIB 代理、大量 JSP），报错 `OutOfMemoryError: Metaspace`。调优参数：`-XX:MetaspaceSize` 初始大小、`-XX:MaxMetaspaceSize` 最大大小。静态变量和 static final 常量也存储在此区域，其中编译期常量（如 `static final int B = 20`）在准备阶段就完成赋值。
