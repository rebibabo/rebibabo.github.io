# 类加载

[[wiki/glossary/java-basic/index|返回词汇表]]

## 定义

类加载是 JVM 将 `.class` 文件的字节码读取到内存中并生成 `Class` 对象的过程，分为加载、验证、准备、解析、初始化五个阶段。

## 上下文

类的生命周期为：加载 --> 验证 --> 准备 --> 解析 --> 初始化 --> 使用 --> 卸载，其中验证、准备、解析合称"连接（Linking）"。加载阶段读取字节码生成 Class 对象；准备阶段为 static 变量分配内存并赋默认值（0/null/false）；初始化阶段才执行 static 代码块和 static 变量的真正赋值。类加载遵循**双亲委派模型**：类加载器收到请求后优先委派父加载器，逐层向上直到 Bootstrap ClassLoader，保证核心类不被篡改。触发初始化的场景包括 new 对象、访问 static 变量、调用 static 方法、Class.forName 等，但访问 static final 编译期常量不会触发初始化。
