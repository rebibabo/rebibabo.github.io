多态（Polymorphism）

[[wiki/glossary/java-basic/index|返回词汇表]]

定义

多态是面向对象的核心特性，指同一方法调用在不同子类对象上表现出不同行为。在 Java 中，多态通过父类引用指向子类对象（向上转型）+ 运行时动态绑定实现——调用哪个方法取决于对象的实际类型，而不是引用的声明类型。

上下文

只有普通实例方法参与多态，static 方法和字段都不参与（static 方法看引用声明类型，叫”隐藏”不叫重写；字段也在编译时绑定）。多态的典型应用是面向接口/父类编程：方法参数声明为父类型或接口类型，运行时传入不同的子类实现，代码无需修改即可扩展。

向下转型（(Dog) animal）需要用 instanceof 先行检查，否则运行时可能抛 ClassCastException。Java 16+ 支持模式匹配：if (a instanceof Dog dog) 一步完成检查加转型。

相关术语

  • [[wiki/glossary/java-basic/方法重写|方法重写]] — 多态的实现基础，子类重写的方法决定运行时实际调用的行为
  • [[wiki/glossary/java-basic/抽象类|抽象类]] — 抽象类多态：声明为抽象类型，运行时动态绑定到具体子类实现
  • [[wiki/glossary/java-basic/接口|接口]] — 接口多态：声明为接口类型，运行时绑定不同实现类，是最常用的多态形式
