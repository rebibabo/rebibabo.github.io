缓冲流

[[wiki/glossary/java-basic/index|返回词汇表]]

定义

缓冲流（Buffered Stream）是 Java I/O 装饰器模式中的增强型流，通过在内存中设置缓冲区来减少对底层系统调用的次数，从而大幅提升 I/O 操作的性能。

上下文

字节缓冲流：BufferedInputStream 和 BufferedOutputStream，预读取或暂存一批字节数据，减少每次 read/write 触发系统调用的开销。字符缓冲流：BufferedReader 和 BufferedWriter，除了缓冲功能外，BufferedReader 还提供了便捷的 readLine() 方法逐行读取文本，BufferedWriter 提供了 newLine() 写入平台无关的换行符。缓冲流本身不连接数据源，必须包裹在基础流（如 FileInputStream、FileReader）外面，是装饰器模式的典型应用。文件复制时使用 8KB 的 byte[] 缓冲区配合 BufferedInputStream 是经典的高效写法。注意缓冲流写入后需要 flush() 或关闭流才能确保数据真正落盘。

相关术语

  • [[wiki/glossary/java-basic/字符流|字符流]] — 字符缓冲流装饰在字符流外层，提供 readLine/newLine 便捷方法
