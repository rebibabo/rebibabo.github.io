# Bid Response（竞价响应）

[[wiki/glossary/programmatic-ads/index|返回词汇表]]

## 定义

Bid Response 是 DSP 收到 Bid Request 后返回给 SSP/ADX 的竞价响应消息。它告诉广告交易所："我愿意为这次曝光出价 X 元，用这个广告素材"。

## 上下文

Bid Response 的核心字段包括：出价金额（price）、广告素材地址（ad markup）、素材 ID、目标跳转地址。所有出价以 CPM 为计价单位（即千次曝光的价格），ADX 会将所有 DSP 出价换算后比较。

如果 DSP 决定不出价，返回 HTTP 204 No Content（即 No Bid），不需要发回 Bid Response JSON。
